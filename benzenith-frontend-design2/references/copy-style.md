# Copy Style (ZH/EN)

## Bilingual hierarchy
- 中文优先承载情绪与意境；英文作为补充解释（更小、低对比、或分行呈现）。
- 允许英文大标题（serif），但必须维持“书卷气”与留白，不要科技感标题。

## Punctuation & spacing
- 中文段落用全角标点；英文段落用半角标点。
- 中英文混排时，英文词组与中文之间可保留细微间隔（视觉上更清爽）。
- “®” 必须紧贴系列名（例如：随心扇®Series）。

## CTA wording
- 默认 CTA 用短句：如“即刻探索”
- 避免电商化强促销词（如“立刻购买”“限时折扣”），除非业务明确需要。
