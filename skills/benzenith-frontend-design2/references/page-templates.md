# Page Templates (BenZenith v2)

> 每次生成页面/组件，必须选择并声明一个模板。

## Template A: Home (Editorial + Gallery)
Modules order (recommended):
1) Header (minimal, black bar, centered logo, right nav)
2) Hero Banner (water/ink structure + centered text lockup)
3) Hero supporting copy (bilingual editorial block, large serif EN headline optional)
4) Series entry (dark section, 3 series icons/entry, short copy + editorial CTA)
5) Product highlight / grid (4-up or curated set)
6) Brand visual break (light section with product close-up / still-life)
7) Brand history teaser (editorial layout: title + long text)
8) Exhibition teaser (dark section: “觀水知万象” + intro + CTA)
9) Footer signature (links + huge “BenZenith” wordmark)

Layout rules:
- 必须有“黑/浅底交替”的展陈节奏（Ink ↔ Sand/Ivory）。
- 每个模块之间用留白与 hairline 分隔，不用重卡片堆叠。
- CTA 默认是 editorial link（如“即刻探索”）。

## Template B: Series Landing
Modules:
1) Header
2) Series Hero (ink + product hero, title/series name)
3) Intro text (centered, short editorial paragraph)
4) Product grid (4 items typical; card minimal)
5) Footer signature

Grid rules:
- 产品网格：优先 2–4 列（desktop），移动端 1 列。
- 卡片：图片主导，文字轻、低对比；hover 仅做下划线/微光。

## Template C: Product Detail (Gallery split)
Modules:
1) Header
2) Split hero: Left = large product image, Right = title + status + intro text
3) Status: “Coming Soon” (or stock state)
4) Long description (editorial paragraph)
5) Footer signature

Rules:
- 右侧文字必须是“长文可读”的 editorial 排版（line-height 足够）。
- 交互克制：不堆叠复杂规格选择 UI；除非用户明确需要电商购买流程。

## Template D: Brand Story (Left visual + Right text)
Modules:
1) Header
2) Two-column layout: Left = ink/brand visual, Right = long-form story
3) Optional timeline blocks
4) Footer signature

Rules:
- 这是“阅读页”，不能做成营销 landing 的花哨卡片。

## Template E: Exhibition / Event (Poster + Minimal info)
Modules:
1) Header (dark)
2) Poster left + details right
3) Editorial CTA
4) Footer signature

Rules:
- 黑底为主，信息极简、像展览海报墙。

## Footer Signature Module (mandatory style)
Composition:
- Link groups (left or upper area): “珠宝系列 / 高級定制 / 品牌动向 / 关于我们 / 联系我们 / …”
- Newsletter email input（若需要）
- Huge wordmark: “BenZenith” 超大字号作为版式锚点（可被裁切，允许溢出）

Rules:
- 这不是普通 footer；必须具备“品牌签名墙”的视觉存在感。
