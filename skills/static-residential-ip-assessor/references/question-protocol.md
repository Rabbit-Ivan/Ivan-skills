# Question Protocol

在开始调查前，用最少问题锁定目标。

## Quick Confirmation（最多 1 轮）

当用户提供 IP 时，快速确认：

1. 你给的是哪种输入？
   - 单个 IP
   - IP 段（CIDR）
   - ASN + Hostname + Range

2. 你最看重哪件事？
   - A. 判断是否静态家宽住宅
   - B. 判断是否会被风控拦截
   - C. 两者都要

## Fast Start Rule

满足以下任一条件，**跳过提问直接开始分析**：

1. 用户直接发送了 IP 地址（无其他问题）→ 使用默认值立即开始
2. 用户在消息中已说明目的（如"帮我查一下这个 IP 能不能用"）→ 使用默认值立即开始
3. 用户回复了上述问题 → 按回答开始

## Default Values

未指定时采用以下默认值：

- 目标：双目标（属性 + 风险）
- 严格度：保守（账号安全优先）
- 输出：可审计报告（含来源和时间戳）
- 灰区放行：允许（需二次验证）
- 复查周期：14 天
